Создание модели

Принимает POST-запросы

Регистрация

Доступна по адресу /login

Принимает POST-запросы

В теле запроса имя пользователя, которому присваивается ключ API и данные записываются в бд.

Обновление модели

Доступна по адресу /models/:id

Принимает PUT-запросы

В теле запроса модель,выбираемая по ip, которую мы хотим изменить.

Удаление модели
